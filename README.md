# yolo-poseモデルを用いたRyze Telloの自律接近
### 背景   
* 安価な構成でドローンの自律制御をやってみたい
* 自律追跡は簡単にできたけど，せっかくだしもっといろいろやってみたい
* 姿勢推定モデル使って、接近する部位を指定出来たら面白そう

### 機材構成
* Raspberry Pi 5
* Hailo-8
* Ryze Tello
* ホストとなるPC
* Raspberry Pi 5用のネットワークアダプタ

### 動作例
マネキンを対象にした実験を実施予定(実施後掲載)
